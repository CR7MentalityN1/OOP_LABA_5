# Лабораторная работа №5. Практика ООП

Проект содержит решения заданий по ООП:
- Класс Product (Товар)
- Класс FuelTank (Топливный бак)
- Класс Elevator (Лифт)
- Класс LotteryBall (Лототрон)
- Класс Employee (Сотрудник)

## Запуск

1. Откройте проект в Visual Studio или VS Code.
2. Соберите и запустите файл src/Program.cs.
