# Лабораторная работа №5. Практика ООП

## Содержание
1. **Product** — класс товара с выводом информации.
2. **FuelTank** — топливный бак с заправкой и расходом.
3. **Elevator** — лифт с перемещением между этажами.
4. **LotteryBall** — лототрон для генерации случайных чисел.
5. **Employee** — сотрудник с уровнем навыка и энергией.

## Запуск
1. Откройте проект в Visual Studio или VS Code.
2. Соберите и запустите файл src/Program.cs.
